# Introduction to Applied Regression Analysis

**ICPSR Summer Program in Quantitative Methods of Social Research**

**Instructor:** Patrick Shea, University of Glasgow

**Dates:** May 26–29, 2026

This repository contains course materials — slides, R code, and data — for the ICPSR course *Introduction to Applied Regression Analysis*. All materials are in the [ICPSR_OLS26](./ICPSR_OLS26) folder.

## Software

This course uses R and RStudio. Both are free and available for all major operating systems. Please install them before the first session:

- [Download R](https://www.r-project.org/)
- [Download RStudio](https://posit.co/download/rstudio-desktop/)

No prior experience with R is assumed.
